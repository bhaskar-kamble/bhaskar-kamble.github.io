Zuerst möchte ich einfach einen Grafen plotten. Die Funktion, die ich plotten möchte ist

\begin{equation} f(x) = \frac{\sin x}{x}. \end{equation}

Unten ploote ich diese Funktion, wie auf dieser Link beschrieben wird.

import matplotlib.pyplot as plt
import numpy as np
pi = np.pi
x = np.linspace(-4*pi,4*pi,1000)
plt.rcParams['figure.figsize'] = (11,4)
plt.rcParams.update({'font.size' : 17})
plt.yticks([0,1],['$0$','$1$'])
plt.xticks([-4*pi,-2*pi,0,2*pi,4*pi],['$-4\pi$','$-2\pi$','$0$','$2\pi$','$4\pi$'])
plt.ylim(-0.5,1.5)
plt.plot(x,np.sin(x)/x,label=r'$f(x)=\frac{\sin x}{x}$')
plt.legend(loc="best",fontsize=20)
plt.show()

YouTube Videos¶

Hier ist eine Methode, die ich vor einer Zeit gelernt habe, mit deren man YouTube Videos hochladen kann. Erstmal soll man das Video auf YouTube besuchen. Unter dem Video gibt es ein "SHARE" button. Wenn man dort klickt, öffnet sich ein Window mit verschiedenen Optionen auf Facebook, Twitter etc. hochzuladen. In diesem Window, unten Rechts, gibt es "EMBED" und "COPY" Buttons. Wenn man auf "EMBED" klickt, öffnet sich ein neues Window mit einem Embedcode. Diesen Code muss man kopieren und in zwischen inverted commas ('xyz') in Klammern einfügen - die Klammern, die nach HTML kommen. Cell Type muss "Code" sein.

from IPython.display import HTML
HTML('embed code')
